Release Log

Patch 0.8.2

Mobile
PC

Patch 0.8.2 adds the civilian ID to the civ details.

- Quick addition of the civilian ID to the details when clicking on a persona.
- Allows users to view the associated ID when clicking on their civilian.

Patch 0.8.1

Mobile
PC

Patch 0.8.1 fixes search box for first and last name. 💪

- A couple fixes where we now do a regex search for first and last name
- Allows users to start typing when writing a ticket or warning and we will do an active search on the police database
- Prevents form from being submitted if user does not click on dropdown. (We need the ID to assign the ticket to the correct civilian)
- Minor other bug fixes and code cleanup

Release 0.8.0

Mobile
PC

Release 0.8.0 adds the ability to display civilians with the same name. 🙌

- When searching for civilians, displays them each as separate records.
- When writing a citation you can now assign to a specific civilian, even if there are duplicates
- Adds new dropdown when typing the civilian name for assigning a ticket
- Adds new dropdown when typing the civilian name for writing a warning
- When writing a citation you can now assign to a specific civilian, even if there are duplicates
- To work with backwards compatibility, if we are unsure of where the tickets or warnings should be assigned, if there are duplicate names assigned to a ticket we will default to the first user that was created. Moving forward all new citations and warnings will be assigned to specific civilians based on the civilian ID.

Patch 0.7.1

Mobile
PC

Patch 0.7.1 adds ability to modify Civilian vehicle details.
Now you can click on a specific civilian vehicle and modify the details.
Modify the Plate, Model, Color, Registration and Insurance Status and Stolen Status.
Now if you need to update details on a specific vehicle you can do so.

Release 0.7.0

Mobile
PC

Release 0.7.0 adds ability to modify Civilian details.
Now you can click on a specific civilian persona and modify the details. Modify the First or Last Name, Date of Birth, License and warrant Status.
Now if you misspell someone's name you can fix it.
Thank you all for the continued support! You are all the real MVPs 😄

Release 0.6.0

Mobile
PC

Release 0.6.0 allows officers to issue written warnings.
We have heard the feedback and we delivered. Officers can now issue written warnings for civilians. These will now show up in a separate table when searching for a civilian in the name database.

Patch 0.5.1

Mobile
PC

Patch 0.5.1 adds support to delete a specific civilian.
We fixed how we were deleting civilians so now it is not possible to delete other peoples civilians by accident.
Separates the name input when writing tickets into First Name and Last Name. We will have more info for this in future releases, but for now it allows us to more easily associate a ticket with a civilian.
Lastly, contains some small bug fixes.

Release 0.5.0

Mobile
PC

Release 0.5.0 displays multiple vehicles when searching by license plate.
Up until now we only supported returning a single vehicle by license plate.
When searching a license plate you will see all the vehicles with that plate (and we mean all the vehicles).
We also added in some new icons to help identify if a vehicle has valid/invalid Registration or Insurance.
